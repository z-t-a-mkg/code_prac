# PHP8 上級アプリ仕様（ルートA）

## 概要
- 認証/権限/CRUD/検索/ログ/セキュリティ/テストを含む上級課題。

## 要件
- 認証: ログイン/ログアウト
- 権限: ロール別の閲覧/編集制御
- CRUD: 基本的な作成/更新/削除/一覧
- 検索: 条件検索（AND/ORの簡易対応）
- ログ: 重要操作の監査ログ
- セキュリティ: XSS/CSRF/入力検証
- テスト: 最低限の検証スクリプト

## 非機能
- 依存は最小（素のPHP）
- DBは段階的に（JSON → SQLite → MySQL）
