# 課題ID: MODERN-001

## 概要
フォーム → 型バリデーション → 送信（APIはモック可）→ 結果表示。

## 難易度 / 所要時間
- 難易度: Medium
- 所要時間: 30-90分

## テーマ
- React、TypeScriptの型、バリデーション、状態管理

## 受け入れ条件
1. 型: 入力データの型が明示されている
2. 型: 必須項目の未入力が型またはバリデーションで検出される
3. 型: 文字数制限が型/バリデーションで検出される
4. 正常系: 送信成功で結果が表示される
5. 異常系: 送信失敗でエラーメッセージが出る
6. UI: 入力中のエラー表示がある
7. 型: APIレスポンスの型が定義されている
8. 品質: lint/typecheck が通る

## 手順（3ステップ）
- Step1: 最小動作（成功1ケース）
- Step2: 異常系（3ケース）
- Step3: セキュリティ/品質（入力検証/責務分離）

## チェック方法
- `npm run typecheck` / `npm run lint`
