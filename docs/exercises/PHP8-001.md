# 課題ID: PHP8-001

## 概要
フォーム入力 → サーバ側バリデーション → JSON保存 → 一覧表示（HTMLエスケープ必須）。

## 難易度 / 所要時間
- 難易度: Medium
- 所要時間: 30-90分

## テーマ
- フォーム、バリデーション、JSON保存、XSS対策

## 受け入れ条件
1. 正常系: 必須項目が全て入力されていれば保存される
2. 正常系: 保存後に一覧に表示される
3. 異常系: 必須項目が空ならエラーになる
4. 異常系: 文字数制限を超えたらエラーになる
5. 異常系: 形式不正（例: メール）ならエラーになる
6. 異常系: JSON保存に失敗したらエラーになる
7. セキュリティ: 一覧表示はHTMLエスケープされる
8. セキュリティ: 簡易CSRF対策のトークンチェックがある

## 手順（3ステップ）
- Step1: 成功1ケースの保存と表示
- Step2: 異常系3ケースのバリデーション
- Step3: XSS/CSRFなどのセキュリティ対策

## チェック方法
- `php8/tests/run.sh`
